# SkillSale Print Bridge

Android WebView APK that opens SkillSale staff UI and prints queue numbers.

- Package: `co.skillsale.print`
- Firebase: `app/google-services.json` (FCM ready)
- Splash: animated logo on `#151918` before WebView

## Build

```bash
./gradlew assembleDebug
```

APK: `app/build/outputs/apk/debug/app-debug.apk`
